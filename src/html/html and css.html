Introduction to HTML and CSS

HTML is short for Hyper Text Markup Protocol. ( It’s one of those acronyms people forget the full meaning because the HTML acronym is so popular.) HTML is the code used to structure a webpage and its content; It’s the standard markup language for creating Web pages.

HTML Tags

HTML tags are keywords that define how your web browser will format and display content. HTML tags also help web browsers distinguish between HTML content and simple content.

They are usually enclosed in these brackets "< >" Most HTML tags consist of an opening bracket, content, and a closing bracket For example:

""

As you can see we first have the opening bracket “<” Then the content "tag" followed by the closing bracket “>”

When you use an open tag you usually have to close the tag except in some cases. For Example:

" ... "

As you can see we first have the opening tag "" Then the ending tag "", which has a slash before the "tag" content unlike the opening content. Don’t worry about what’s between the ellipses it would be explained in the next section

HTML Elements

HTML Elements are what make up an HTML file. These elements are responsible for creating web pages and define content in that webpage. An HTML element consists of a start tag, some content, and then an end tag Like this:

"Hello world"

As seen in this example we have the starting tag "" then some content "Hello world" and finally, the end tag “”

HTML Attributes

HTML Attributes are special words that provide additional information about elements. They are specified in the start tag and come in name/value pairs like name=” value ” For example:

"content"

As seen in this example we have the starting tag '“” Some content and then the end tag “”

Block elements are the elements that structure the main part of a web page, by dividing a page into coherent blocks. Block elements always start with new a line and take up the full width of the webpage available from left to right. Browsers automatically add some space (a margin) before and after the element. CSS Rule

A CSS rule set contains a selector and a declaration block.

"h1{ color: red; font-size: 12px;}"

In the code above the “h1” is the selector The “color” is the property “red” is the value “font-size” is also a property “12px” is also a value And everything in the curly bracket is the “declaration block”

To explain the sections above: Selector: The Selector indicates the HTML element you want to style. For example, the

tag is used to define a paragraph in an HTML document as explained in the Block element chapter.

Declaration Block: The declaration block can contain one or more declarations separated by a semicolon. For the above example, there are two declarations:

1. color: red;

2. font-size: 12px;

Each declaration contains a property name and value, separated by a colon.

Property: A Property is a type of attribute of an HTML element. Some examples are color, border, etc.

Value: Values are assigned to CSS properties. In the above example, the value "red" is assigned to the color property.
